## SAMPLE FRONTEND APP

> Built with:
- NextJs
- TailwindCSS
- Apollo-client

Conected to a graphql project, this frontend has CRUD features: 
- Users can create a record
- Users can Update, Get, Remove records
- It also has a simple pagination feature to navigate between records
